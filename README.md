# ربات جستجوگر بله

این ربات به شما امکان می‌دهد که متن مورد نظر خود را در گوگل جستجو کنید و نتایج را مستقیماً در پیام‌رسان بله دریافت کنید.

## ویژگی‌ها

- 🔍 جستجوی خودکار در گوگل
- 🤖 پاسخگویی سریع به پیام‌های کاربر
- 📱 رابط ساده و کاربرپسند

## نصب و راه‌اندازی

### 1. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 2. دریافت توکن ربات

1. در پیام‌رسان بله به `@BotFather` پیام دهید
2. دستور `/newbot` را ارسال کنید
3. نام و نام کاربری ربات خود را وارد کنید
4. توکن ربات را دریافت کنید

### 3. تنظیم متغیرهای محیطی

فایل `.env.example` را کپی کرده و به `.env` تغییر نام دهید:

```bash
cp .env.example .env
```

سپس توکن ربات خود را در فایل `.env` وارد کنید:

```
BALE_BOT_TOKEN=your_actual_bot_token_here
```

### 4. اجرای ربات

```bash
python bot.py
```

## نحوه استفاده

1. ربات خود را در بله پیدا کنید
2. دستور `/start` را ارسال کنید
3. هر متنی که می‌خواهید جستجو کنید را ارسال کنید
4. منتظر نتایج بمانید!

## مثال

```
کاربر: هوش مصنوعی چیست؟
ربات: 🔍 در حال جستجو...
ربات: نتایج جستجو برای "هوش مصنوعی چیست؟":
1. [عنوان نتیجه اول](لینک)
2. [عنوان نتیجه دوم](لینک)
...
```

## نکات مهم

- مطمئن شوید که توکن ربات به درستی در فایل `.env` قرار گرفته است
- برای استفاده در سرور، می‌توانید از ابزارهایی مثل `screen` یا `tmux` استفاده کنید
- برای deployment در production، استفاده از webhook به جای polling توصیه می‌شود
