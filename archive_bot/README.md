# archive_bot (Standalone)

This folder is a standalone project for the Bale archive/analysis bot.

## Run

1. Copy env template:
   - `cp .env.example .env`
2. Fill required keys in `.env`.
3. Start:
   - `bash start.sh`

Or manually:

- `python3 -m venv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`
- `python bot.py`
